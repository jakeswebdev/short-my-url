URL-Shortener API MicroService.

How to use?

  • To shorten a new URL, simply pass it as a parameter to our URL this way: OUR_URL/new/THE_URL_YOU_WANT_TO_SHORTEN
    • Example: https://shortmyurl.glitch.me/new/https://www.youtube.com
  • You'll get a JSON formatted response with your original url and the shortened version.
    • Example:{"long_url":"https://www.youtube.com","short_url":"shortmyurl.glitch.me/ByydYuQaG"}
